# Папка media

Идея:
В этой папке находятся и "загружённые" и "хранимые" фотографии. Сделано сцелью иммитации загрузки 
файлов на удалённый сервер. Чтобы "загрузить новое" изображение необходимо сначала в распакованном 
проекте, в папку media положить файл, который Вы "загружаете". Затем, при редактировании профиля, Вы 
"загружаете" из этой же папки на "удалённый сервер". Название файла сохраниется в localStorage, что 
в итоге создаёт впечатленеие произошедшей агрузки файла. Удаление происходит путём удаления информации 
о существовании этого файл(в car.photoхъ).
